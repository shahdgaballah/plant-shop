# GreenMind - Plant Shop Landing Page

A modern and responsive landing page for a plant shop, designed to showcase products and attract customers.

## Features

- Clean and modern landing page design
- Responsive navigation with cart, profile, and menu icons
- Hero section with compelling headline and search functionality
- Best selling plants showcase with pricing
- About us section highlighting key business features:
  - Large assortment of plants
  - Fast & Free shipping
  - 24/7 customer support
- Interactive elements with smooth hover effects
- Mobile-friendly layout

## Technologies Used

- HTML5
- CSS3
- Font Awesome Icons for navigation and UI elements
